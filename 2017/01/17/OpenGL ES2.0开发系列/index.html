title:OpenGL ES2.0开发系列

##1.基本概念：
1.OpenGL 绘制的都是图形，包括形状和填充，基本形状是三角形。
2.每个形状都有顶点，Vertix，顶点的序列就是一个图形。图形有所谓的正反面，如果我们看向一个图形，
它的顶点序列是逆时针方向，那我们看到的就是正面。
3.Shader(着色器),用来描述如何绘制(渲染)，GLSL是 OpenGL的编程语言，全称就叫 OpenGL Shader Language。OpenGL渲染需要两种shader，vertex 和 fragment。
3.1 Vertex shader，控制顶点的绘制，指定坐标、变换等。
3.2 Fragment shader，控制形状内区域渲染，纹理填充内容。

##2. 坐标系：
　 　2.1 手机坐标系: 二维坐标系 原点在左上角,x轴向右递增,y轴向下递增 (X,y取值范围为屏幕的分辨率)
2.2 OpenGL坐标系: 三维坐标系 原点在中间,x轴向右，y轴向上,z轴朝向我们 (x,y,z的取值方位都是[-1,1])
　　 2.3 纹理坐标系: 二维坐标系 原点在左下角,x轴向右,y轴向上 (x,y的取值范围都是[0,1])
